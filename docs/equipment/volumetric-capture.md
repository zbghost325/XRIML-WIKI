# Volumetric Video Capture

<div class="page-meta">
  <span class="meta-date">Last updated: 2026-03-30 &middot; v1.0</span>
  <span class="tag">Equipment</span>
  <span class="tag">Volumetric Capture</span>

3D volumetric recording systems for capturing real-world subjects as viewable 3D content.

## Available Systems

| System | Type | Location | Notes |
|--------|------|----------|-------|
| Azure Kinect DK | Depth sensor | Lab use only | Time-of-flight depth + RGB, body tracking SDK |
| Intel RealSense D455 | Depth sensor | Lab use only | Stereo depth, lightweight |
| Depthkit | Software + multi-sensor | Lab use only | Volumetric video production pipeline |

## Use Cases

- Recording 3D performances for VR playback
- Creating volumetric assets for Unity/Unreal projects
- Research in spatial computing and telepresence

!!! warning "Training Required"
    Volumetric capture setups require orientation and scheduling. Contact the lab manager to arrange access.
